Experimental means bounded

What it does not claim.

Not a complete person Chats are partial, selected, relationship-specific evidence.

Not autonomous The system cannot send, promise, transact, or represent the owner.

Not finished Automatic typed-state extraction and complete deletion propagation remain incomplete.

Not an official API WhatsApp for Mac is version-sensitive; wacli is a third-party mirror.

Private by default

Build something you can inspect before something that can imitate.

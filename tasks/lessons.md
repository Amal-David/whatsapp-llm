# Lessons

- When the persona workbench is framed as "best output" or "authentic human texting," do not stop at a small export UI. The expected bar is a broad research-backed recommender with source coverage, code-switching/paralinguistic guidance, low-data augmentation strategy, and explicit authenticity diagnostics.
- When the user corrects a docs cleanup target, prioritize the newly named public reference and remove stale metadata before broadening the cleanup.
